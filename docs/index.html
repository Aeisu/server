Workflow

base: All new work should begin life as a copy of this branch, regardless of its destination at completion time. Be sure to base your pull requests off of this base branch!

everything : Everything really means everything! All things, if they have been reviewed, can and will arrive at this branch - including things in the base branch. If we don’t think something is ready for the base branch, it can still get into the everything branch!

others: there may be additional branches not specified here.
You should not ever pull them to your server, you WILL eventually get a broken server from it.
We will NOT help you with that. You have been warned!
